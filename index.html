About This Site
This site is a hand-written static website — just index.html plus two CSS files (colors_and_type.css for the design tokens and typography, and mobile.css for the responsive layout), with no build step, bundler, or JavaScript framework in the way.
Source lives in a GitHub repo (shirazik/khs-website) and is deployed via GitHub Pages, which serves the files straight from the main branch; a CNAME file in the repo points the deployment at the custom domain kareemshirazi.com, so every push to main is effectively a production release.
